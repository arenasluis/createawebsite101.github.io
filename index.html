How to create a web page using Github 101

1.Sign up for a free account at github.com

2.Once you've signed up, click on "create a new repository"

3.Choose a name for your website! And by the message I got it's always better if you keep it short :)

4.Choose whether you'd like this site to be public or private.

5.I chose to add the README file here but I wasn't told it was neccesary.

Click "Create repository

6.On the next page, click on "Add file" and "upload files."

7.Drag your files and drop them here. (Make sure to include a index.html file with your code.)

8.Click on commit changes and watch the magic happen!(Sometimes you need to wait for a little while)
